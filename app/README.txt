https://vk.com/dev/goods_docs (VKAPI)
Порядок установки 
1. Создание приложения для вк https://vk.com/editapp?act=create
Нам понидобится приложение для сайта.
В настройках (вкладка Настройки) необходимо указать Доверенный redirect URI
- https://[адрес скрипта]/app/auth.php
- https://[адрес скрипта]/app/yaml2vk.php
2. Настройки самого скрипта. Файл settings.php
Обычно достаточно указать 
- CLIENT_ID -- ID приложения находятся на странице настроек приложения вкладка Настройки
- CLIENT_SECRET -- Защищённый ключ находятся на странице настроек приложения вкладка Настройки
- ID -- ID группы с которой будет работать скрипт. Узрать может администратор на странице настроек группы. Например из Номер сообщества — club191399337 получится 191399337
- _ID -- -ID группы с которой будет работать скрипт. Узрать может администратор на странице настроек группы. Например из Номер сообщества — club191399337 получится -191399337
- REDIRECT_URL - https://[адрес скрипта]/app/auth.php
- REDIRECT_AFTER_AUTH - https://[адрес скрипта]/app/yaml2vk.php

3. Получение токена
для получения токена необходимо запустить https://[адрес скрипта]/app/auth.php и подтвердить права приложения
! ВК не проверяет права админа на этом этапе и никак не дает знать о их необходимости
Токен записывается в token.dat и не имее срока годности

4. После получения токена можно выполнить тестовый запуск с главной страницы скрипра
пароль по умолчанию Iddqd2011

5. CRONTAB
Понадобится добавить две записи в сронтаб для работы скрипта
- GET https://[адрес скрипта]/app//yaml2vk.php?key=Iddqd2011 -- для ЕЖЕДНЕВНОЙ выгрузки
- GET https://[адрес скрипта]/app/timer.php -- ЕЖЕМИНУТНЫЙ свспомогательный скрипт для регру
